※ 파일구조

- css

  • /lib/jquery-ui-1.12.1/jquery-ui.min.css : jquery ui css
  • /lib/slick/slick.css : slick css
  • /font/font.css : 폰트
  • /css/reset.css : 기본설정
  • /css/common.css : 공통부분 ( header, GNB, container, footer, popup 등 )
  • /css/component.css : 컴포넌트 ( button, input, selectBox, tab 등 )
  • /css/template.css : 템플릿 ( table, list 등 )
  • /css/contents.css : 전체 페이지

- js

  • /lib/jquery/jquery-3.3.1.min.js : jquery 3.3.1 버전
  • /lib/jquery-ui-1.12.1/jquery-ui.min.js : jquery ui 1.12.1 버전 (jquery 1.7+ 이상 4.0 미만 지원 )
  • /lib/slick/slick.js : 슬라이드 라이브러리 slick js
  • /js/uicomponent.js : ui component js

- fonts

  • KBFGTextL : KB금융 본문체 Light
  • KBFGTextM : KB금융 본문체 Medium
  • KBFGTextB : KB금융 본문체 Bold
  • KBFGDisplayL : KB금융 제목체 Light
  • KBFGDisplayM : KB금융 제목체 Medium
  • KBFGDisplayB : KB금융 제목체 Bold

- html

  • /cmm/ : 공통
  • /ast/ : 내자산
  • /buy/ : 내차구매
  • /tec/ : 차테크
  • /mycar/ : 내차고
  • /adm/ : 관리자

※ 네이밍 규칙

  • html 파일명 : 화면 ID 와 동일 하게. 단 점(.) 은 언더바(_) 로 대체

※ 주석 규칙

  • html : 태그 시작 지점 <-- 내용 --> , 태그 종료 지점 <-- // 내용 --> ,

※ CSS 규칙

  • css class name : 단어간 구분은 대시(-)로 구분
  • css code style : component, template 을 제외한 화면 ID 를 시작으로함 예) .A_CL_MA_0010 .section-01>ul>li
    => 화면ID는 모든 페이지의 <div class="content A_CL_MA_0010"> 에 class로 들어감
  • UIComponent 적용 유무 : UIComponent 실행이 된 Element 에 initiate 클래스 추가(없을경우 미적용)

※ JAVASCRIPT 규칙

  • javascript version : ES5
  • function name : 카멜 표기법, 소문자의 동사 예) getValue()
  • 변수 name : 소문자의 명사 예) var value;
  • 상수 name : 대문자의 명사 예) var CONSTANT_VALUE;

※ fixed z-index 계층

  • .CMYMCP132 .btn-floating : 10
  • .btn-area bottom-fixed : 100
  • .content-header : 1000
  • .popup-container : 5000
  • .toast.message : 6000
